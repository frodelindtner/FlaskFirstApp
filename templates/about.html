Det indholder to sider til at vise resultater for Baseball liga'er

  • USA - data kommer fra API
  • Danmark - local - data ligger i local SQL database

Det er ikke muligt at ændre data, som kommer fra API'et - der bliver leveret af SportDataIO.
Men det er muligt at ændre og opsætte data til den lokale danske liga.
Bemærk at når jeg fanger data fra externt API skal der bruges en gyldig API KEY.

Jeg har benyttet FLASK framework "3-layer seperation".

Præsentation (HTML CSS) JINJA2 Templates Route's
Forretnings logic Services Methods Objekter
Data adgang SQLite Connector Mapping

Ved at benytte denne arkitektur adskiller man ansvaret for diverse componenter i websitet.
Hvilket giver bedre overblik, og bedre muligheder for genanvendelse af logik.
